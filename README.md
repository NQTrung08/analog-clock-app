# React + Vite

1. clone project from github: https://github.com/NQTrung08/analog-clock-app.git

    ```cmd
      git clone: https://github.com/NQTrung08/analog-clock-app.git
    ```

2. Install node_modules

    ```cmd
      npm install
    ```

3. Launch project

    ```cmd
      npm run dev
    ```
